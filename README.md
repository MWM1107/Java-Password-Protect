# Password Protect

Password Protect is a Java application that provides a comprehensive solution for evaluating password strength and checking for exposure to data breaches. It includes a graphical user interface (GUI) for easy interaction, allowing users to upload CSV files containing passwords for analysis. 

## Features:

- **Password Strength Evaluation**: Assess password length, complexity, and uniqueness.
- **Data Breach Checking**: Utilizes the "Have I Been Pwned" API to check if any passwords have been compromised in data breaches.
- **Graphical User Interface**: Easy-to-use interface for interacting with the application.
- **Password Generation**: Generate strong passwords with customizable length.

## How to Use:

1. Download the Password Protect Java application.
2. Run the application.
3. Use the GUI to upload a CSV file containing passwords for analysis.
4. Receive comprehensive feedback on password strength, uniqueness, and exposure to data breaches.
5. Generate strong passwords as needed.

## Requirements:

- Java Development Kit (JDK) 8 or higher
- Internet connection for data breach checking

## Installation:

1. Clone the repository.
2. Compile the Java source files.
3. Run the compiled application.
